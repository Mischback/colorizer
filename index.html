Colorizer

A tool to build color palettes while considering contrast values.

RGB input
Red component
Green component
Blue component

The RGB input lets you specify the values for R, G and B components of the color directly.

Each of the components may be specified in a range between 0 and 255.

The values may be entered directly or using the corresponding sliders.

Oklch input
Lightness component
Chroma component
Hue component

Oklch is the polar form of the Oklab colorspace.

It lets you specify the (perceived) Lightness (L), Chroma (C) and Hue (H).

While Lightness and Chroma are specified in percent-based values in range [0..100], the Hue is an angle in range [0..360].

The values may be entered directly or using the corresponding sliders.
