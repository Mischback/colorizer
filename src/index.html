Colorizer

A tool to build color palettes while considering contrast values.

Color Palette

    Add a new Color

    RGB input
    Red component
    Green component
    Blue component

    The RGB input lets you specify the values for R, G and B components of the color directly.

    Each of the components may be specified in a range between 0 and 255.

    The values may be entered directly or using the corresponding sliders.

    HSL input
    Hue component
    Saturation component
    Light component

    HSL is a different notation for the sRGB colorspace.

    It lets you specify the Hue (H), Saturation (S) and Light (L).

    While Saturation and Light are specified in percent-based values in range [0..100], the Hue is an angle in range [0..360].

    The values may be entered directly or using the corresponding sliders.

    HWB input
    Hue component
    White component
    Black component

    HWB is a different notation for the sRGB colorspace.

    It lets you specify the Hue (H), Whiteness (W) and Blackness (B).

    While Whiteness and Blackness are specified in percent-based values in range [0..100], the Hue is an angle in range [0..360].

    The values may be entered directly or using the corresponding sliders.

    Oklch input
    Lightness component
    Chroma component
    Hue component

    Oklch is the polar form of the Oklab colorspace.

    It lets you specify the (perceived) Lightness (L), Chroma (C) and Hue (H).

    While Lightness and Chroma are specified in percent-based values in range [0..100], the Hue is an angle in range [0..360].

    The values may be entered directly or using the corresponding sliders.
